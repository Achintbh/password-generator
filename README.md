CLI Password Generator
Overview
This is a straightforward Python command-line tool that generates strong, secure passwords. The generated passwords include a mix of letters (both uppercase and lowercase), digits, and special characters to ensure high security.

Features
Customizable password length based on user input

Supports uppercase and lowercase letters

Includes numeric digits

Incorporates special symbols for extra complexity

Randomizes the order of characters to enhance password security

Installation
Ensure Python 3.x is installed on your machine.

Clone this repository to your local system.

Usage
Run the program, then enter the desired count of letters, digits, and special characters when prompted. The program will generate a randomized password that meets your specifications.

Example Session
text
Welcome to the password generator
Enter the number of letters for your password: 5
Enter the number of digits for your password: 3
Enter the number of special characters for your password: 2
